# Intranet Acessanet

Sistema de intranet corporativa desenvolvido para centralizar ferramentas, sistemas e informações internas da empresa **Acessanet**.

A aplicação permite que colaboradores acessem rapidamente sistemas internos, dashboards operacionais, documentação técnica e links importantes utilizados no dia a dia da empresa.

## Objetivo

Centralizar o acesso a recursos internos da empresa, melhorando a organização das informações e aumentando a produtividade das equipes técnicas e administrativas.

## Tecnologias utilizadas

Frontend
- React
- JavaScript
- HTML5
- CSS3

Backend
- Node.js
- Express

Banco de Dados
- MySQL

## Funcionalidades

- Acesso rápido a sistemas internos
- Centralização de links e ferramentas da empresa
- Dashboard com informações operacionais
- Interface responsiva
- Organização de recursos internos
